## 架构设计（轻量级版）

### 技术栈
- 后端：FastAPI + SQLite
- 前端：Vue.js + Element Plus + Vite
- 爬虫引擎：Requests + BeautifulSoup4 + Playwright（按需使用）

### 功能模块
1. **爬虫任务管理**：创建任务、配置规则、任务调度、监控
2. **数据处理**：数据清洗、转换、导出(CSV/JSON/Excel)、预览
3. **API接口**：RESTful API、批量数据获取
